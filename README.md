📘 GLOWING Codex — Official Specification & Core Data Schema

A Web3-native human insight protocol for persistent identity, narrative encoding, and long-term digital continuity

🧩 Overview

GLOWING Codex 是一套以 Web3 思維打造的「人類洞察協議（Human Insight Protocol）」。
它的使命是將「個體的天賦、生命原型、行為模式、靈魂路徑」等深層資訊，用一致而可解讀的方式編碼，形成一個 可延伸、可驗證、可跨世代保存 200 年的數據結構。

Codex 本身不依賴特定平台。
它可以被儲存在：

IPFS

Arweave

Lighthouse

任何 JSON-friendly 的資料湖

甚至是單純的 Git repository

使資料具有高度可攜性與未來相容性。

🎯 Mission Statement

建立一個“能跨越世代”的個體資料協議。
不受平台壽命影響，不依賴單一公司，能夠在 10 年、50 年、200 年後仍被解讀。

GLOWING Codex 旨在成為：

靈性系統的跨文化共通語言

命盤研究框架的標準化資料層

人格結構與原型分析的開源協議

Web3 身分（Web3 Native Identity）的底層資訊

長期敘事、心理模型、個體本質的統一描述法

🧱 Core Concepts
1. Profile Archetype（人生角色原型）

每個人僅有 12 類 Archetype，並可分為：

兒童型態

青年型態

中年型態

並自動具備：

男性版 / 女性版（外觀風格差異，但不影響功能）

2. Soul Digits（靈魂數字結構）

每一組結構由：

腦袋（Head）

身體（Body）

整合訊息（Integration）

所組成，並且採用：

全滿 / 半滿 / 全空（能量狀態）

工具數（外層 Tool Digit）

內部工具群（Inner Digits）

目的數（Final Digit）

3. Codex Schema（資料結構）

所有原型、數字結構與敘事都將以：

JSON Schema

YAML Schema

TypeScript Interface

共同定義。

以確保：

前端可以直接 render

後端可以直接驗證

AI 可以直接引用

Web3 儲存時不會破壞結構
